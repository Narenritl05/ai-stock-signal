เปิดหรือปิดเครื่องมีผลอย่างไร

  • ถ้าเปิดคอม/mini PC ทิ้งไว้ Telegram bot แบบถามตอบจะทำงานทันที, ระบบรันตามเวลาที่ตั้งไว้ได้, watchdog/self-heal ตรวจทุก 30 นาทีได้ และหน้าเว็บ local จะเปิดดูจากเครื่องหรือในวง LAN ได้
  • ถ้าปิดเครื่อง bot ที่รันบนเครื่องจะหยุดตอบ, งาน local ที่ตั้งเวลาไว้จะไม่รัน และเว็บ local เช่น http://localhost:8000/ จะเข้าไม่ได้จนกว่าเปิดเครื่องใหม่
  • GitHub Actions ยังช่วยได้ งานที่ตั้งไว้บน GitHub ยังรันตามรอบฟรีได้ถ้า repo เปิดใช้งานอยู่ และสามารถส่ง Telegram digest/อัปเดต GitHub Pages จากข้อมูลล่าสุดได้โดยไม่ต้องเปิด notebook
  • ข้อมูลบนเว็บไม่หาย ไฟล์ใน docs/data จะยังเป็นข้อมูลรอบล่าสุดที่บันทึกไว้ แต่อาจเก่าลงถ้าไม่มีเครื่องหรือ GitHub Actions ช่วยรันอัปเดต

เปิดเว็บเพื่อดูข้อมูลทั้งหมดได้ไหม

  • ได้ หน้าแรกโหมด ทุกตลาด รวมหุ้นไทยและหุ้นต่างประเทศไว้แล้ว ใช้ช่องค้นหาเพื่อดูรายชื่อหุ้นทั้งหมดใน watchlist ได้
  • เปิดจากเครื่องนี้ รันเว็บ local แล้วเข้า http://localhost:8000/ เพื่อดู dashboard, สัญญาณ, backtest, สถานะระบบ และข้อมูลรายตัว
  • เปิดจากมือถือในบ้าน ถ้าใช้ mini PC เป็น server ให้เปิดเว็บแบบ LAN ด้วย http://IP-ของ-mini-PC:8000/ โดยไม่ต้องเปิด port ออกอินเทอร์เน็ต
  • เปิดจากนอกบ้าน ใช้ GitHub Pages จะปลอดภัยกว่าและไม่มีค่า Cloud เพิ่ม แต่จะเห็นข้อมูลรอบล่าสุดที่ GitHub Actions สร้างไว้ ไม่ใช่หน้าเว็บ local สดจากเครื่อง
  • ดูข้อมูลรายตัว กดการ์ดหุ้นในหน้าเว็บเพื่อดูราคา, สัญญาณซื้อ/ขาย, จุดเข้า, stop loss, เป้า, risk/reward, volume, ATR, ข่าว และกรอบถือสั้น/ยาว

⚠️ ข้อควรรู้ก่อนใช้

  • ไม่มีระบบใดการันตีกำไร 100% — เครื่องมือนี้เป็น "ตัวช่วยคัดกรอง" เพื่อประหยัดเวลา ไม่ใช่เครื่องพิมพ์เงิน
  • ไม่ใช่คำแนะนำการลงทุน และ ไม่สั่งซื้อ-ขายแทนคุณ — การตัดสินใจสุดท้ายเป็นของคุณ
  • ควรตั้งจุดตัดขาดทุน (stop loss) เสมอ และลงทุนเท่าที่รับความเสี่ยงได้
  • ข้อมูลจาก Yahoo Finance เป็น end-of-day/ดีเลย์ เหมาะกับการวางแผนรอบถัดไป

📚 อธิบายศัพท์ (สำหรับมือใหม่)

คะแนนสัญญาณ (0–100)คะแนนรวมจากหลายอินดิเคเตอร์ ยิ่งสูงยิ่งน่าสนใจ — แต่ไม่ใช่การการันตี
RSIวัดว่าหุ้น "ซื้อมากไป (>70)" หรือ "ขายมากไป (<30)" ช่วยดูจังหวะกลับตัว
MACDวัดโมเมนตัม/แรงส่ง ถ้าเป็นบวกและเพิ่มขึ้น = แรงซื้อกำลังมา
EMA20 / EMA50เส้นค่าเฉลี่ยราคา ถ้าราคายืนเหนือเส้น = แนวโน้มขาขึ้น
วอลุ่ม (Volume)ปริมาณการซื้อขาย ถ้าพุ่งสูงกว่าปกติ = มีแรงเข้าจริง
Stop Loss (ตัดขาดทุน)ราคาที่ตั้งไว้ว่า "ถ้าหลุดจะขายทันที" เพื่อจำกัดการขาดทุน — ต้องมีทุกไม้!
เป้าหมาย (Target)ราคาที่วางแผนจะขายทำกำไร
ขนาดไม้ (Position Size)ซื้อกี่หุ้น คำนวณจากความเสี่ยงที่รับได้ ไม่ใช่ซื้อสุดตัว
ภาวะตลาด (Breadth)สัดส่วนหุ้นที่ขาขึ้น บอกว่าตลาดรวมดีหรือแย่ — ไม่ควรสวนตลาด

✅ กฎทอง 5 ข้อของมือใหม่

  • ตั้ง stop loss ทุกไม้ — ก่อนซื้อต้องรู้ว่า "ผิดแล้วออกตรงไหน"
  • เสี่ยงไม่เกิน 2% ต่อไม้ — ไม้เดียวพังต้องไม่ทำพอร์ตเจ๊ง
  • อย่าไล่ราคา — พลาดไม้นี้ยังมีไม้หน้า อย่า FOMO
  • ฝึกด้วยเงินปลอมก่อน — จนชนะสม่ำเสมอค่อยใช้เงินจริง
  • ไม่มีอะไรการันตีกำไร — เป้าหมายคือ "ขาดทุนน้อย อยู่รอดนาน"

🆓 ฟรี 100% ไม่มีค่า server

GitHub Actions (รันอัตโนมัติ) + GitHub Pages (โฮสต์เว็บ) + Yahoo Finance (ข้อมูล) + Telegram (แจ้งเตือน) — ฟรีทั้งหมด
